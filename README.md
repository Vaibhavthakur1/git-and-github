# Final Project - Getting Started with Git and GitHub

This repository is the final project for the Coursera course **"Getting Started with Git and GitHub"**.

## Project Description
This project demonstrates basic Git and GitHub usage including cloning, branching, committing, merging, and submitting a pull request. It also includes a Bash script to calculate simple interest.

## Files Included
- `simple-interest.sh` → Bash script to calculate simple interest.
- `LICENSE` → Apache 2.0 License.
- `CODE_OF_CONDUCT.md` → Code of conduct for contributors.
- `CONTRIBUTING.md` → How to contribute to this repository.

1. Open terminal.
2. Navigate to the project directory.
